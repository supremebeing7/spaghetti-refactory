Lehman tags: - rails - ruby - metaprogramming modified_time: '2015-12-26T23:20:40.674-08:00' blogger_id: tag:blogger.com,1999:blog-4459370175540830177.post-3720759093811998158 blogger_orig_url: http://spaghettirefactory.blogspot.com/2015/12/funky-ruby-class-method-syntax.html --- I’ve run across this syntax a few different times but never grasped how it worked, and I guess never cared enough to look it up. Turns out it’s dead simple.

This:

class Person  
  def self.species
    "Homo Sapien"
  end
end

is equivalent to this:

class Person  
  class << self
    def species
      "Homo Sapien"
    end
  end
end

Thanks to Yehuda Katz for finally clearing that up for me.

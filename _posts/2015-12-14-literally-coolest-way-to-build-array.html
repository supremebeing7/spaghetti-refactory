Lehman tags: - rails - ruby modified_time: '2015-12-26T23:14:44.430-08:00' blogger_id: tag:blogger.com,1999:blog-4459370175540830177.post-3056161514067136744 blogger_orig_url: http://spaghettirefactory.blogspot.com/2015/12/literally-coolest-way-to-build-array.html --- One of Ruby’s literal syntax shorthands, %i essentially loops over each word, converts to a symbol and adds it to an array. For example:

%i(dog cat mouse)
# => [:dog, :cat, :mouse]

Caveat: This syntax was added in Ruby 2.0, so don’t go trying it in any previous version.
